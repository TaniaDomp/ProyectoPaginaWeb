Libreria

Este lugar es perfecto para relajarte con el aire de tranquilidad propio de una librería.

Cuenta con varias secciones que se encuentran ubicados desde la planta baja y dos pisos, es el lugar perfecto para que disfrutes de un buen libro.

Cafeteria

Hay una cafetería para que disfrutes una bebida o un postre mientras lees.

Puedes tener un desayuno, comida o cena y después tomar un café mientras decides cuál libro llevarte a la mesa.

Área de esparcimiento

Si no quieres estar en la cafetería, pide tu café para llevar y ve al área de sillones. Además, en el segundo piso se ubica una sección de cine para niños y cine de culto que también se aprovecha para pequeñas obras teatrales.

De la decoración seguro amarás el techo y las paredes repletas de libros.
